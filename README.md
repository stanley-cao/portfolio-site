# https://stanleycao.vercel.app/
welcome to v2 of my personal website!

## running instructions
clone the repo
```
git clone https://github.com/stanley-cao/portfolio-site.git
```

cd into the project folder
```
cd portfolio-site
```

install dependencies
```
npm install
```

run
```
npm run dev
```
